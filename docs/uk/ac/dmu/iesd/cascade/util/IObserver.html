public interface IObserver

Interface IObserver defines how an IObserver implementer should behave by basically implementing the update method, so that those objects wanting to be informed about changes in observable objects get notified/updated.

update

public void update(Object obs, Object arg)
This method is called whenever the observed object is changed. An application/object/agent calls an Observable object's notifyObservers method to have all the object's observers notified of the change.
Parameters:
obs - the observable object.
arg - an argument passed to the notifyObservers method.
