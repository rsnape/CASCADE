uk.ac.dmu.iesd.cascade.util
Class ArrayUtils

public class ArrayUtils

A helper class with static methods to aid in the manipulation of arrays IMPLEMENTATION NOTE - (except where explicitly stated) methods result in an array as output, the methods do not work directly on the array(s) passed in but rather create a new array and pass this back as the return value.
