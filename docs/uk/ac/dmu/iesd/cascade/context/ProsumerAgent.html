uk.ac.dmu.iesd.cascade.context
Class ProsumerAgent

Direct Known Subclasses:
GeneratorProsumer, HouseholdProsumer, NonDomesticProsumer, StorageProsumer, HHProsumer
public abstract class ProsumerAgent
implements ICognitiveAgent

A ProsumerAgent is an object which can both consume and generate electricity at the same time. The ProsumerAgent class is the abstract superclass of prosumer agents. Examples of prosumer agents are the household and wind generator agents.
